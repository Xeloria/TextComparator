Instructions for TextComparator

1) Make sure the folders "COMPARE_TEXT" and "RESULTS" are located
   in the same directory as the program.

2) Place two text files that are to be compared inside the folder
   "COMPARE_TEXT" and write the text file names in "TEXT_FILE_NAMES.txt".
   
3) Run the program. The result shall be generated and located in
   "RESULTS" folder.
   
4) Results interpretation:
	Lines beginning with "#| " are different in both files.
	Lines beginning with " | " are exactly the same in both files.
